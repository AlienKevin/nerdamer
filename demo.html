Instructions

To enter an expression just type it in the box below with the format: expression;variable=value e.g. cos(x)+1,x=PI
To enter a known value type in: number_of_expression: variable=value, variable2=value. e.g. %1, x=5, y=6 . Constant $\pi$ = PI and e = E

To set a variable type in variable:value e.g. x:8

To declare a function type in: function_name( variables_separated_by_commas ):=function_expression e.g. f(x):=x^2-5/cos(x). To access the last declared expression use % or use %[digit] to access a previously entered expression.

Math is rendered using Mathjax.
Equations are calculated using nerdamer.js
Equation input is done with EqInput
